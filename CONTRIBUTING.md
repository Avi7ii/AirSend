# 🤝 参与贡献 (Contributing to AirSend)

首先，感谢你对 AirSend 的关注！我们非常欢迎各种形式的贡献。

## 🛠 开发环境

1.  **系统需求**: macOS 13.0+
2.  **开发工具**: Xcode 15.0+ 或 Swift 6.2+
3.  **构建脚本**: 项目根目录下提供有 `build.sh`，可快速构建并运行项目。

## 📝 贡献流程

### 1. 提交 Bug 或需求
请在 GitHub Issues 中提交。在提交前，请先搜索是否已存在类似的 Issue。

### 2. 提交代码 (Pull Request)
*   Fork 本仓库。
*   在你的分支上进行修改。
*   提交 PR 前，请确保代码能够通过编译。
*   请在 PR 描述中清晰地说明你的修改动机和改动。

## 🎨 样式规范
*   遵循 Swift 官方 API 设计指南。
*   UI 部分优先使用原生视觉元素。
